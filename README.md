# 📚 Library Management System (Python)

A simple **Library Management System** built using Python that allows users to manage books and users, and perform operations like borrowing and returning books.

---

## 🚀 Features

- Add new books 📖
- View all books
- Search books by title
- Search books by author
- Add users 👤
- Borrow books
- Return books
- Track available books
- Save book data to file
- Load book data from file

---

## 🛠️ Technologies Used

- Python (Core Programming)
- Object-Oriented Programming (OOP)
- File Handling

---

## 📂 Project Structure

- LibraryItem → Base class for library items
- Book → Inherits from LibraryItem
- User → Handles user operations
- Utility functions:
  - Search books
  - Validate ISBN
  - Manage users
  - File handling

---
## ⚙️ How to Run

1. Clone the repository using:
`git clone https://github.com/ashutoshkashyap04/Library-Management-System`

2. Go to the project folder:
`cd Library_Management_System`

3. Run the program:
`python main.py`
